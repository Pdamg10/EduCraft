# EduCraft — Servicios Académicos 🎓

> *"Tu éxito académico, construido contigo."*

Sitio web estático profesional para **EduCraft**, un servicio académico que ofrece apoyo en trabajos escritos, presentaciones, infografías, programación, mapas mentales, matemáticas y más.

---

## 🌐 Demo

Abre `index.html` directamente en tu navegador o despliega con cualquier hosting estático (GitHub Pages, Netlify, Vercel, etc.).

---

## 📁 Estructura del proyecto

```
EduCraft/
├── index.html      # Estructura HTML de la página
├── style.css       # Estilos (CSS puro, sin frameworks)
└── script.js       # Funcionalidad JavaScript (vanilla)
```

---

## ✨ Secciones

| Sección | Descripción |
|---|---|
| **Hero** | Llamada a la acción principal con animación de badge |
| **Servicios** | Tarjetas filtrables por categoría (escritura, programación, visual, ciencias) |
| **Cómo funciona** | Proceso de 4 pasos para contratar el servicio |
| **Precios** | Tablas de precios por categoría + planes mensuales |
| **FAQ** | Acordeón de preguntas frecuentes |
| **Contacto** | Formulario de solicitud + info de WhatsApp y correos |

---

## 🛠️ Tecnologías

- **HTML5** semántico
- **CSS3** con variables personalizadas (design tokens)
- **JavaScript** vanilla (ES6+)
- **Font Awesome 6** para íconos
- **Google Fonts** — Inter + Poppins

No se usa ningún framework ni dependencia de Node.js. Basta con abrir el `index.html`.

---

## 🎨 Paleta de colores

| Token | Valor | Uso |
|---|---|---|
| `--morado-principal` | `#6C3FC5` | Color de marca |
| `--morado-oscuro` | `#3B1F7A` | Hero, navbar, footer |
| `--dorado` | `#C9A84C` | Acentos, CTAs, precios |
| `--morado-suave` | `#EDE7FF` | Fondos secundarios |

---

## ⚙️ Funcionalidades JavaScript

- **Navbar** con efecto scroll + blur glassmorphism
- **Menú móvil** con animación hamburger
- **Filtro de servicios** por pestaña/categoría
- **Tabs de precios** con animación de panel
- **Acordeón FAQ** con transición de altura
- **Scroll reveal** con `IntersectionObserver`
- **Smooth scroll** en todos los anclas
- **Validación de formulario** en tiempo real
